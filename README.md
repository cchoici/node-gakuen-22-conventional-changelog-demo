# cc-demo

A Vue.js Plugin



## Build Setup

    # install dependencies
    npm install

    # serve with hot reload at localhost:8080
    npm run dev

    # lint source codes
    npm run lint

    # build for production with minification
    npm run build

    # run unit tests
    npm run unit

    # run all tests
    npm test
